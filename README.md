# DevBot — AI Coding Assistant 👾

## 🌐 Live Demo
👉 https://web-production-e2cff.up.railway.app

## 💻 GitHub
👉 https://github.com/lasya0809/devbot

DevBot is an AI-powered coding assistant that helps developers 
debug errors, understand code, and learn new concepts through 
a simple conversational interface.

## ✨ Features
- AI-Powered Answers via Groq API + Llama 3.3 70B
- Multi-Turn Conversations with memory
- Code Block Rendering
- Typing Indicator
- Suggested Starter Prompts
- Clean Dark UI

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python, FastAPI |
| AI Model | Llama 3.3 70B |
| AI Provider | Groq API |
| Frontend | HTML, CSS, Vanilla JS |
| Deployment | Railway |

## ⚙️ Setup
1. Clone: `git clone https://github.com/lasya0809/devbot`
2. Install: `pip install -r requirements.txt`
3. Set key: `export GROQ_API_KEY=your_key_here`
4. Run: `python main.py`
5. Open: `http://localhost:8000`
